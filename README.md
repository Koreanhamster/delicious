## 프로젝트 목적과 의의
- (본인이) 영어와 요리를 둘 다 좋아하기에, 영어사용자들에게 아시안 레시피를 소개하는 서비스 구상
- 오픈API 사용하여 데이터를 가져오고 필요한 데이터를 적절히 사용하는 것에 의의

### 사용 스킬
- React
- Styled-components
